# Random Quote Generator

A simple command-line application that prints a random quote loaded from a text file.

## How it works
- Quotes are stored in a text file
- The program selects one quote randomly
- Output is printed to the console

## Structure
- src/ – application logic
- quotes/ – quote storage
- tests/ – unit tests
- .github/workflows – CI pipeline
